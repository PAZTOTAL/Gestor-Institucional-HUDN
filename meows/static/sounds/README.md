# Sonidos de Alerta MEOWS

Esta carpeta contiene los archivos de audio para las alertas sonoras del sistema MEOWS.

## Archivo requerido:

- **alert_soft.mp3** o **alert_soft.wav**: Sonido suave y corto (0.8-1.2 segundos) que se reproduce cuando se detecta un RIESGO ALTO (ROJO).

## Características del sonido recomendado:

- ✅ Tipo: tono corto, grave, suave
- ✅ Duración: 0.8 – 1.2 segundos
- ✅ No debe ser estridente
- ✅ No debe saturar al personal

## Sugerencias de búsqueda:

- "soft alert tone hospital"
- "subtle notification sound"
- "gentle hospital alert"

## Nota:

El sistema solo reproduce el sonido cuando:
1. El riesgo cambia a ALTO (ROJO)
2. Es una nueva alerta (no se repite si el riesgo ya era ALTO)
3. El navegador permite la reproducción de audio

Si no se coloca el archivo de sonido, el sistema funcionará normalmente sin reproducir sonido.
